# CargoManagementSystem

Cargo Management System is used to order cargo.  
Admin can login into cargo management system for booking the cargo and other purposes. 
Admin can mention about their  items  in  cargo  management system once they log in check which company contains it. 
Also, admin can give administration rights to desired users. 
Database entities like Customer, Order, Product, Shipper, Supplier, etc. can be modified easily.
This allows users to access information about the cargo quantity handled within the company warehouse. 
This web application allows users to easily access all cargo related information and order the required cargo through admin. 
The main objective of this product is to systematically store cargo related information and modify manually.
